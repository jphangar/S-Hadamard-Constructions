# Butson Hadamard Matrix Equivalence Package

This is a copy of the source code for the BH equivalence algorithm implemented by Egan, Flannery and Ó Catháin.
The source code is available at <https://www.daneflannery.com/classifying-cocyclic-butson-hadamard-matrices>.

We include a copy of the source code to make it easier to run code given in our repository.
This is not our original work nor have we made any noteworthy changes to the original package.

Please note that the package is implemented in Magma.
The primary functions of interest are given in `ButsonFunctions.m`, please see this file for more details.
